Projeto de estudo de uma loja virtual (e-commerce) desenvolvido para praticar desenvolvimento web, organização de código e versionamento com GitHub.

## 📌 Sobre o projeto
O **Devstore Project** é uma aplicação web que simula o funcionamento de uma loja virtual, permitindo visualizar produtos e entender a estrutura básica de um e-commerce.

Este projeto foi criado com o objetivo de aprimorar conhecimentos em desenvolvimento web e boas práticas de programação.

## 🚀 Tecnologias utilizadas
As principais tecnologias usadas no projeto:

- HTML
- CSS
- JavaScript
- Git e GitHub

Devstore-Project

- Exibição de produtos
- Layout de loja virtual
- Interface responsiva
- Interações com JavaScript
